# joscales
Ardy project to work with scales massa-k via rs232

involves RTC and pump control to fill the weighted liquid

connection: rs socket - colour - ardypin:
2 - orange 	- 11
3 - red		- 10
5 - brown	- GND

This is used for RTC
adafruit RTClib fork of jeelabs fantastic lib

note: default linux arduino that is in the repo is garbage
